>  采用harbor 搭建企业级私服

## 下载harbor
```shell
wget https://github.com/goharbor/harbor/releases/download/v2.1.3/harbor-offline-installer-v2.1.3.tgz
```

## 安装 harbor
- 首先准备安装包
```shell
./prepare
```
- 执行安装程序
```shell
./install
```
