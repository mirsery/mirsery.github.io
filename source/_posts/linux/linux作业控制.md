##显示作业列表 jobs
 在任何时候可以使用jobs命令来显示所有作业的列表。该命令的语法为：
 ```bash
 jobs [-1]
 ```
 在大多是情况下，只需要输出命令本身:
 ```bash
 jobs
 ```
 如果需要查看作业的进程ID 以及作业号和命令名，则可以使用-l选项
```bash
 jobs -l
 ```
 如果不指定作业号这些指令在默认情况下会作用在当前作业。在大多数情况下

##将作业提交至前台: fg
当需要将作业移至前台时，可以使用fg命令，使用该命令有下列三种形式:
```bash
fg
fg % [job]
% [job]
```
作业号|含义
---|---
%%|当前作业
%+|当前作业
%-|前一个作业
%n|作业#n
%name|含有指定名的作业
%?name|命令中任意位置含有name的作业

##将作业移至后台 bg
```bash
bg [%job]
```
