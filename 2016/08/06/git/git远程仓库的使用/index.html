git远程仓库的使用

作者: mirsery 创建日期: 2016年8月6日

查看当前的远程仓库

git remote
git remote -v   (v是verbose的缩写)

添加远程仓库

git remote add [shortname][url]

从远程仓库抓取数据

git fetch[remote-name]
该命令会拉取远程仓库中的内容到本地

推送数据到远程仓库

 git push origin master

查看远程仓库信息

git remote show
