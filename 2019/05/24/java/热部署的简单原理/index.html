简单介绍

在类加载器中，类只可以加载一次并且不可卸载。java类只能被加载一次，并且无法卸载。想要实现热部署可以分以下三个步骤：
1、销毁该自定义ClassLoader
2、更新class类文件
3、创建新的ClassLoader去加载更新后的class类文件。
