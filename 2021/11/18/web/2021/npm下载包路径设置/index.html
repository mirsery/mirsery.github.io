MAC npm 下载包路径设置

查看npm的prefix和cache路径配置信息

npm config get cache
npm config get prefix

设置下载路径 缓存路径

npm config set cache "/cachePath"
npm config set prefix "/prefixPath"

查看npm配置信息

npm config list

清空npm缓存

npm cache clean -f

验证缓存数据的有效性和完整性，清理垃圾数据

npm cache verfy
