java运算符

2021年7月12日

运算符是一种特殊的符号，用来表示数据的运算，赋值和比较等。

算术运算符

四则混合运算： + , -, x , /

取模运算：% , 取模运算的结果和被模数的正负相关。例如 -5 % 3 = -2

赋值运算符

+=, -= ,/= , *= ,%= , =

逻辑运算符

&, |, !, &&, ||, ^

位运算符

位运算符操作的都是整型数据

  • << 1 左移，相当于*2(正数高位补0，负数补1)
  • >> 1 右移，相当于**/2** (正数高位补0，负数补1)
  • >>> 无符号右移，不管此整数正负高位都补0
